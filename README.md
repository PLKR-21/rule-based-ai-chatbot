# 🤖 Rule-Based AI Chatbot

A simple rule-based chatbot built using Python as part of the DecodeLabs Artificial Intelligence Industrial Training Program (Project 1).

## Features

- Continuous conversation loop
- Handles greetings
- Uses dictionary-based responses
- Input sanitization using `lower()` and `strip()`
- Default response for unknown inputs
- Exit commands (`bye`, `exit`, `quit`)

## Technologies Used

- Python 3
- Dictionaries
- While Loop
- If-Else Logic
